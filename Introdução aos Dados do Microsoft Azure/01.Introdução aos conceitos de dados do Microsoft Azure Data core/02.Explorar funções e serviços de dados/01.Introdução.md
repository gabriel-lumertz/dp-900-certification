# Introdução

Na última década, a quantidade de dados gerados por sistemas e dispositivos aumentou consideravelmente. Devido a esse aumento, novas tecnologias, funções e abordagens de trabalho com os dados estão afetando os profissionais de dados. Os profissionais de dados normalmente desempenham diferentes funções ao gerenciar, usar e controlar dados. Neste módulo, você aprenderá sobre as várias funções nas quais as organizações costumam usar profissionais de dados, as tarefas e responsabilidades associadas a essas funções e os serviços do Microsoft Azure usados para realizá-las.
