# Resumo

O Azure dá suporte a uma variedade de serviços de banco de dados que você pode usar para dar suporte a novos aplicativos de nuvem ou migrar aplicativos existentes para a nuvem.

Neste módulo, você aprendeu a:

- Identificar opções para serviços SQL do Azure
- Identificar opções para bancos de dados de software livre no Azure
- Provisionar um serviço de banco de dados no Azure

## Próximas etapas

Agora que você aprendeu sobre os serviços de banco de dados relacional do Azure, considere aprender mais sobre cargas de trabalho relacionadas a dados no Azure buscando uma certificação da Microsoft nos [Conceitos Básicos de Dados do Azure](https://learn.microsoft.com/pt-br/certifications/azure-data-fundamentals/).
