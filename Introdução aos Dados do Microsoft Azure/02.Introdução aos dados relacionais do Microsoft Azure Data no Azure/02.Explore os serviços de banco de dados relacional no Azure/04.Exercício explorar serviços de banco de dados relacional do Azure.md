# Exercício: explorar serviços de banco de dados relacional do Azure

Escolha seu banco de dados

- Banco de Dados SQL do Azure
- Banco de Dados do Azure para PostgreSQL
- Bando de Dados do Azure para MySQl

Agora é sua oportunidade de explorar serviços de banco de dados relacional no Azure.
