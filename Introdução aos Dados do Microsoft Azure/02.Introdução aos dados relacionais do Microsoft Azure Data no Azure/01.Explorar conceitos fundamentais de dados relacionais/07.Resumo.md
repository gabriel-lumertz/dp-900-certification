# Resumo

Bancos de dados relacionais são uma maneira comum de aplicativos transacionais armazenarem e gerenciarem dados. Eles consistem em um esquema de *tabelas*, que são vinculadas por meio de valores de chave comuns. Você usa o SQL para consultar e manipular os dados nas tabelas e pode enriquecer o banco de dados criando objetos como exibições, procedimentos armazenados e índices.

Neste módulo, você aprendeu a:

- Identificar características de dados relacionais
- Definir normalização
- Identificar tipos de instrução SQL
- Identificar objetos de banco de dados relacionais comuns
