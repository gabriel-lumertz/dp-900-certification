# Introdução

A maioria dos aplicativos de software precisa armazenar dados. Muitas vezes, os dados são armazenados em um banco de dados relacional, no qual os dados são organizados em tabelas relacionadas e gerenciados usando a linguagem SQL. No entanto, muitos aplicativos não precisam da estrutura rígida de um banco de dados relacional e dependem de armazenamento não relacional (geralmente chamado de NoSQL).

O Armazenamento do Microsoft Azure e o Microsoft OneLake oferecem uma variedade de opções para armazenar dados na nuvem. Neste módulo, você explorará os recursos fundamentais do Armazenamento do Microsoft Azure OneLake e aprenderá como ele é usado para dar suporte a aplicativos que exigem armazenamentos de dados não relacionais.
