# Resumo

O Azure Armazenamento é um serviço fundamental no Microsoft Azure e permite uma ampla gama de cenários e soluções de armazenamento de dados.

Neste módulo, você aprendeu a:

- Descrever recursos e funcionalidades do armazenamento de Blobs do Azure
- Descrever recursos e funcionalidades do Azure Data Lake Gen2
- Descrever recursos e funcionalidades do Microsoft OneLake
- Descrever recursos e funcionalidades do armazenamento de arquivos do Azure
- Descrever recursos e capacidades do armazenamento de tabelas do Azure
- Provisionar e usar uma conta de armazenamento do Azure

## Próximas etapas
