# Introdução

Os bancos de dados relacionais armazenam dados em tabelas relacionais, mas, às vezes, a estrutura imposta por esse modelo pode ser muito rígida e geralmente ocasiona um baixo desempenho, a menos que você gaste tempo implementando ajustes detalhados. Existem outros modelos, coletivamente conhecidos como bancos de dados *NoSQL*. Esses modelos armazenam dados em outras estruturas, como documentos, grafos, repositórios de chave-valor e repositórios de famílias de colunas.

O Azure Cosmos DB é um serviço de banco de dados de nuvem altamente escalonável para dados NoSQL.
