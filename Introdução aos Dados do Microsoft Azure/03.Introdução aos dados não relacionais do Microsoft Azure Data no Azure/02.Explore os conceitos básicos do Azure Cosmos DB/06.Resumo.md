# Resumo

o Azure Cosmos DB fornece uma solução de banco de dados de escala global para dados não relacionais.

Neste módulo, você aprendeu a:

- Descrever os principais recursos e funcionalidades do Azure Cosmos DB
- Identificar as APIs do Azure Cosmos DB
- Provisionar e usar uma instância do Azure Cosmos DB

## Próximas etapas

Agora que você aprendeu sobre o Azure Cosmos DB para armazenamento de dados não relacionais, considere aprender mais sobre cargas de trabalho relacionadas a dados no Azure, buscando uma certificação da Microsoft em [Princípios Básicos de Dados do Azure](https://learn.microsoft.com/pt-br/certifications/azure-data-fundamentals/).
